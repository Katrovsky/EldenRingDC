# Elden Ring Death Counter

OBS-оверлей для стримов: имя персонажа и счётчик смертей поверх игры. Дизайн — **Bahamamam**.

---

## Принцип работы

Читает бинарный файл сохранения `ER0000.sl2` с интервалом 500 мс, пропуская итерации без изменения `mtime`. За один проход считываются три небольших региона через `ReadAt` — весь файл (~29 МБ) в память не загружается.

Обновления счётчика доставляются в браузер через SSE без поллинга. Оверлей добавляется в OBS как источник «Браузер» по адресу `http://localhost:8080` (порт настраивается).

---

## Установка и настройка

При первом запуске — интерактивный мастер: автопоиск файла сохранения, выбор слота персонажа, порт сервера. Конфиг сохраняется в `config.json` рядом с исполняемым файлом.

### Пути к файлу сохранения

#### Windows

```txt
%APPDATA%\EldenRing\<SteamID64>\ER0000.sl2
```

#### Linux (Proton)

Поиск ведётся по стандартным корням Steam и всем библиотекам из `libraryfolders.vdf`:

```txt
<steam_root>/steamapps/compatdata/1245620/pfx/drive_c/users/steamuser/AppData/Roaming/EldenRing/<SteamID64>/ER0000.sl2
```

При нестандартном расположении — задать путь явно в `config.json`:

```json
{
  "save_path": "/path/to/ER0000.sl2"
}
```

## Кастомизация оверлея

Стили — в `index.html`, блок `<style>`. Основные параметры класса `.char`:

| Свойство                             | Описание               |
| ------------------------------------ | ---------------------- |
| `font-size: clamp(24px, 8vw, 120px)` | Размер текста          |
| `font-weight: 600`                   | Жирность               |
| `-webkit-text-stroke: 0.8px #7a6507` | Цвет и толщина обводки |
| `filter: drop-shadow(...)`           | Тени текста            |

Горизонтальные линии — псевдоэлементы `::before` / `::after` у `.death-container`.
